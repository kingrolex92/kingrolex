## 🎮 **Usage Guide**

### **Navigation**
- Use the sidebar to switch between different sections
- Click on any menu item to navigate to that section
- The header shows the current section and live update status

### **Dashboard**
- Monitor real-time statistics and system health
- View recent bookings and system status
- Toggle live updates on/off as needed
- Access quick actions through the floating action button

### **Creating Trips**
1. Click "Create Trip" button or use the floating action button
2. Select route, bus, departure time, and pricing
3. Submit to create the trip
4. View the new trip in the trips grid with real-time occupancy

### **Managing Bookings**
1. Navigate to the Bookings section
2. Use search and filters to find specific bookings
3. Click action buttons to edit, view, or generate tickets
4. Monitor booking status and payment information

### **Fleet Management**
1. View all buses in the fleet grid
2. Click on seat maps to see availability
3. Manage bus status and specifications
4. Monitor fleet utilization and maintenance schedules

### **System Monitoring**
- Check real-time system metrics
- Monitor service health and connectivity
- View performance indicators and alerts
- Track integration status for all services
